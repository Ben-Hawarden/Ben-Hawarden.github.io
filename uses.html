/uses

What I use day to day for security work, development, and general computing.

// hardware

💻
ThinkPad X1 Carbon
Main machine. Lightweight, durable, excellent keyboard. Running Windows 11 + WSL2.
🖥️
Custom Desktop
Ryzen 7, 32GB RAM, RTX 3070. Used for hashcat GPU cracking and heavier VMs.
🖵
27" 1440p IPS
Main monitor. 1440p makes a big difference for multi-pane terminal work.
⌨️
Keychron K2
Brown switches. Great tactile feedback without the clack. Compact 75% layout.
🖱️
Logitech G Pro
Lightweight, symmetrical. Good enough for everything from CTFs to daily use.

// operating systems

🪟
Windows 11 + WSL2
Daily driver. WSL2 (Ubuntu) for all terminal work. Best of both worlds.
🐧
Kali Linux
Pentesting VM via VMware. Has everything preinstalled, saves setup time.
🧬
REMnux + FlareVM
Isolated VMs for malware analysis. REMnux for Linux samples, FlareVM for PE.
🔒
Whonix
When privacy matters. Tor-routed workstation/gateway VM pair.

// editor & terminal

📝
VS Code
Primary editor with vim keybindings. Remote-WSL extension for seamless Linux dev.
🖥️
Windows Terminal + tmux
Windows Terminal for tabs, tmux inside WSL for pane splits and session persistence.
🐚
zsh + oh-my-zsh
Powerlevel10k theme, zsh-autosuggestions, zsh-syntax-highlighting.
🔤
JetBrains Mono
Ligatures on. Crisp at small sizes. My preferred coding font for years.

// security tools

🔥
Burp Suite Pro
Web application testing. Intruder, Repeater, and the scanner for auth flows.
📡
Nmap / Rustscan
Rustscan for initial discovery, Nmap for service/script enumeration.
💀
Metasploit / Impacket
Exploitation framework and Python SMB/Kerberos/LDAP toolkit. Essential for AD.
🔑
Hashcat
GPU-accelerated password cracking. RTX 3070 makes short work of NTLM hashes.
👻
Ghidra + x64dbg
Reversing toolkit. Ghidra for static analysis, x64dbg for dynamic debugging.
🔍
Amass + Subfinder
Subdomain enumeration for bug bounty recon. Pipe to httpx for live hosts.

// software

🦊
Firefox
uBlock Origin, FoxyProxy, Cookie-Editor. Containers for isolating sessions.
💎
Obsidian
Linked CTF notes, writeup drafts, and a personal wiki. Graph view is excellent.
🔐
KeePassXC
Local password manager. No cloud sync — encrypted .kdbx on personal backup.
🛡️
WireGuard
VPN for HTB / CTF networks. Fast, modern, and trivial to configure.
📦
VMware Workstation Pro
Preferred over VirtualBox. Better snapshot handling and nested virtualisation.
🌿
Lazygit
Terminal git UI. Makes staging hunks and cherry-picking commits painless.
